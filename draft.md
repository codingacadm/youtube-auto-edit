# 무엇을 해야하는가?

1. 유튜브 영상을 가져온다.
- 가져온다?
    - 컴퓨터 하드에 저장한다.
- 어떤 영상을 가져오나?
    - 직접 링크 정리
    - 관심 키워드
    - 조회수, 댓글 등과 같은 조건 (대중적으로 검증되었다고 가정)

2. 편집을한다.
- 편집이 무엇인가?
    - 비디오는 무엇인가?
        - 무엇으로 이루어져 있는가?
            - 소리
            - 자막
            - 타입?
                - 이미지
                - 시간/위치
    - 어떤 행위가 있는가?
        - 시간 구간 잘라나기
        - 배경음 제거/ 말소리만 남김

    - 유익한 부분을 분별하는 방법은?
        - 댓글에 언급이 많은 부분
        - 궁금한 키워드를 포함하는 부분
        - 소리/ 자막 없는 부분

3. 편집 결과 저장한다.
- 하드디스크에 저장을한다.
